분리 집합
=========

서로 공통된 원소를 갖지 않는 집합을 의미한다.  
분리 집합은 두 개 이상의 집합을 일컬을 때만 사용할 수 있는 개념이다.(집합이 하나만 존재한다면 분리되어 있다는 개념이 성립할 수 없음)  
또한 분리되어있기 때문에 교집합이 있을 수 없다.  
그림은 [여기](https://m.blog.naver.com/PostView.nhn?blogId=wnsgh224&logNo=120196308434&proxyReferer=https%3A%2F%2Fwww.google.co.kr%2F)를 가면 나와있다. 

구조
====

우리가 알고 있던 이진 트리에서는 부모가 자식을 가르키고 있었는데 분리 집합에서는 자식이 부모를 가르킨다.  
**[부모에 대한 포인터][데이터]**  
따라서 루트 노드가 집합을 대표(표현)한다.

연산
====

분리 집합에는 두가지 연산만이 존재한다.  
1. 합집합
2. 집합 탐색

## 합집합

두 집합을 합한다.
해당 집합의 대표는 루트 노드이므로 A와 B라는 분리 집합이 존재한다고 가정했을 때  
A와 B를 합한다고하면 B의 루트 노드가 A의 루트 노드를 바라보면 된다.

## 집합 탐색

집합을 탐색한다는 것은 집합에 속해있는 모든 원소(노드)를 탐색한다는 뜻이 아니라  
해당 집합을 대표하는 루트 노드를 찾는 것이다.
